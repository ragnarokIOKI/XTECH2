# XTECH2

Создать приложение в паттерне MVVM, которое содержит одно окно (Window) с элементами: 
 - Кнопка «Печать»
- Текстовое поле (TexBox)
- Метка (Label)
- Таблица с произвольным форматированием
 
При вводе каждого символа в текстовое поле, текстовое поле метки должно изменяться соответствующим образом.
 
При нажатии кнопки «Печать» в произвольном месте формируется pdf-файл  с названием и заголовком из текстового поля и таблицей 1, например «Документ1.pdf»
